ESM系列｜ESM-MSA：蛋白质语言模型MSA Transformer

发表于 2024-05-28 本文字数： 1.7k 阅读时长 ≈ 6 分钟

标题: MSA transformer
期刊/会议: PLMR
年份: 2021

简介

作为Facebook ESM系列的第二篇文章，MSA Transformer于2021年8月发表在PLMR会议上，提出了一个融合多序列比对（MSA, Multi-sequence alignment）信息的新蛋白质语言模型ESM-MSA-1b。不同于以往的蛋白质语言模型使用单个序列进行训练，ESM-MSA-1b有效的利用了蕴含大量进化信息的MSA，加上极度适配MSA信息的轴向注意力机制进行训练，在最少的参数量下达到了当时无监督大语言模型的SOTA水平，在下游任务上表现得也十分出色。

  • ESM-MSA-1b是一个在UR50数据库上训练的掩码语言模型
  • ESM-MSA-1b在三种下游任务：无监督氨基酸接触图预测、监督氨基酸接触图预测和二级结构预测上均达到了当时的SOTA水平

模型概述

问题描述：ESM-MSA-1b是一个掩码语言模型（MLM,Masked Language Modeling），采用随机masking方法。

LMLM(x;θ)=∑(m,i)∈ mask log⁡p(xmi∣x~;θ)\mathcal{L}_{\mathrm{MLM}}(\boldsymbol{x} ; \theta)=\sum_{(m, i) \in \text { mask }} \log p\left(x_{m i} \mid \tilde{\boldsymbol{x}} ; \theta\right) LMLM​(x;θ)=(m,i)∈ mask ∑​logp(xmi​∣x~;θ)

数据集：使用UR50数据库，包含2600万个MSA的大型数据集，平均每个MSA的深度是1192，大小达4.3Tb的数据量

方法：Transformer。比较特殊的是注意力机制采用轴向注意力，使内存占用大大降低，节省计算成本。

O((LM)2)→O(LM2)+O(M2L)O\left((L M)^2\right) \rightarrow O\left(L M^2\right)+O\left(M^2 L\right) O((LM)2)→O(LM2)+O(M2L)

参数量：100M

ESM-MSA-1b模型概述

模型与普通Transformer的区别，其特殊的轴向注意力机制，见下图所示。
对于蛋白质通用规律的学习，MSA信息中同源序列不同位置上的氨基酸对于当前的氨基酸的权重信息并不大，当前氨基酸最重要的关注点还是同一序列其他氨基酸（行）和同源不同序列同一位置的其他氨基酸（列），因此注意力限制在横纵这两条轴向范围就可以了，可以大大降低时间复杂度。

结果

  1. 无监督氨基酸接触预测

    作者采用trRosetta的数据集，在trRosetta数据集中的20个蛋白结构上进行训练，然后以此来预测剩下14842个蛋白质结构数据集。具体而言，是在每个序列的注意力图上训练一个稀疏逻辑回归。这一测试中，MSA Transformer在三种不同序列长度的接触预测上均胜过了Potts模型、ProTrans-T5和ESM-1b模型。在CASP13-FM和CAMEO测试中也是如此。

    ESM-MSA-1b和单序列模型在无监督氨基酸接触预测任务中的平均远程精确度

    作者将用于无监督学习接触图的Transformer拿出来进行分析，发现训练完成的Transformer得到的注意力图与氨基酸接触图息息相关。

    20个结构上训练的稀疏逻辑回归矩阵的权重值

  2. 有监督氨基酸接触预测

    作者用15051个MSA和结构组成的trRosetta 数据集和成对的距离分布进行监督学习。在特征选取上，作者使用了两种方法，一是将sequence embedding作为特征，二是将sequence embedding与对称行注意力图结合，分别进行预测。两种方法在CASP-13FM和CAMEO数据集上的准确率超过了trRosetta,ProTrans-T5和ESM-1b，在L-range上最优为55.8%，L/5-range上最优为80.22%。

    ESM-MSA-1b和其他单序列模型在监督氨基酸接触预测任务中的平均远程精确度

  3. 二级结构预测表现

    作者使用了基于Netsurf结构训练预测8态二级结构的下游任务，比HMM Profile、Protrans-T5、ESM-1b略强，准确率为73.4%左右。

    ESM-MSA-1b和其他单序列模型在CB513数据集二级结构预测上的表现

  4. 错误信息对模型预测的影响

    分别对输入的MSA信息的行和列进行随机打乱的结果。ESM-MSA-1b能在重要信息丢失的情况下依然在氨基酸接触图预测上表现良好。

    ESM-MSA-1b和Potts、ESM-1b被协方差和位置信息干扰后的表现

结果讨论

  • MSA-Transformer输入是多序列比对MSA，而非大量的、并无先验关联的单蛋白质序列，对于语言模型理解蛋白质序列之间的依赖关系有着重要的作用。这种把输入序列的关系先分类好，再用绑定轴向注意力（tied row attention）降低计算成本的方式非常有效。

  • 输入序列多样性对结构预测有很大影响。因为计算成本以及潜在的数据大小不平衡导致的模型表现问题，作者没有把全部的MSA喂给模型，而是选取了其中一部分，作者在四种不同的筛选策略上做了测试，分别是：随机选择序列、汉明距离最大化、汉明距离最小化、HHFilter，其中汉明距离最大化（序列多样性最大）的策略表现最好。这种方式小编认为相当于一种变相的序列去重。
    注意力图与氨基酸接触图是对应的，这很符合直觉上对于蛋白质语言模型中“自注意力”与蛋白质嵌入表示的关联。

  • 比较有意思的是文章在讨论部分测试了更改MSA每一列的值和打乱MSA的列对于Potts模型和ESM-1b的影响，从而阐述MSA列间的协方差和序列位置信息分别对两个模型的干扰，而MSA-transformer在这两个干扰项下还能运作，小编猜测可能与他独特的注意力计算机机制能够从MSA信息中学到协方差和相似性有关，这也使得ESM-MSA-1b比potts模型和ESM-1b的泛化性和鲁棒性更佳。

总结

  • MSA transformer输入是MSA，加入了轴向注意力
  • MSA transformer在预测蛋白质二级结构上表现优于ESM-1b、ProtT5
  • MSA transformer在氨基酸接触图预测上表现优异
  • 将MSA的行列随机打乱，MSA transformer依然表现良好
