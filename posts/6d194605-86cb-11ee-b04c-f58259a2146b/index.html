字典操作

发表于 2023-09-07 更新于 2024-01-06

Python 中字字典常用操作

修改字典的键

⚡️ 只需一行 dic["c"]=dic.pop("a") 即可直接将值弹出到新键

事实上，在Python中，字典的键不能直接修改，因为键是经过哈希处理的。

如果需要修改，可以使用一些技巧间接完成。

方法一(推荐)

弹出要修改的键的值并将其分配给新键

方法二

与方法一类似，使用 update 方法

方法三

首先，将旧键值分配给新键，然后删除旧键。

根据值获取键

方法一

遍历提取键值对，判断值并提取键。

方法二

将字典的值转换为列表，找到目标的索引，将键转换为列表，根据前面的索引获取值。

方法三(不推荐)

新建一个字典，查询字典中的值，但有时值会重复，转成字典后只有一个

将两个列表转换为字典

😎 直接用 dict(zip(list_a, list_b)) 解决，其中 list_a 是键，list_b 是值。

方法一

使用 zip() 和 dict() 将两个列表转换为字典。

方法二

使用字典解析将列表转换为字典。

如果两个列表的长度相同，则可以使用 range() 来遍历字典解析

方法三

使用 for 循环将列表转换为字典。

另一个使用循环的解决方案是使用 range() 迭代索引并检索键值对，就像字典理解的例子一样。
