标题: Generative modeling for protein structures
期刊/会议: NeurIPS
年份: 2018

简介

作者使用 GAN 应用于生成蛋白质结构的任务, 根据蛋白质主链骨架上的 α\alphaα 碳来编码蛋白质结构, 消除了生成模型学习平移对称性和旋转对称性的需要.
其次, 作者引入了一种 3D 结构恢复的凸公式, 以从生成的成对距离图中折叠蛋白质结构, 也可以进行恢复. (使用交替方向乘子法 (ADMM) 将 2D 成对距离“折叠”为 3D 笛卡尔坐标.)
作者提出的算法对于旋转和平移李群 (SE(3)) 中的变换具有不变性(也即三维变换不变性).

本文主要贡献:

  1. 一个蛋白质生成模型, 以SE(3)的方式估计结构分布
  2. 所产生的重建问题的凸公式，我们将其扩展到大型问题实例

RoBERTa 和 ALBERT

发表于 2023-10-25 更新于 2024-01-03 本文字数： 2.1k 阅读时长 ≈ 7 分钟

参考: https://juejin.cn/post/6949901957247172639
RoBERTa论文: 《RoBERTa: A Robustly Optimized BERT Pretraining Approach》
ALBERT论文: 《Albert: A lite bert for self-supervised learning of language representations》

BERT 模型是 2018 年提出的，并在很多自然语言处理任务有前所未有的提升。因此 2019 年就有很多工作是围绕着 BERT 展开的，其中出现了两个 BERT 的改进版模型，RoBERTa 和 ALBERT。
RoBERTa 在更大的数据集和最优的参数中训练 BERT，使 BERT 的性能再次提升。
ALBERT 主要是对 BERT 进行压缩，通过共享所有层的参数以及 Embedding 分解减少 BERT 的参数量。
