## 删除所有出现的元素

### 方法一(推荐)

**使用`filter()`函数从列表中删除元素的所有实例**

1. `filter()` function with `__ne__`
    ```python
    myList = [2, 1, 3, 5, 1, 1, 1, 0]
    myList = list(filter((1).__ne__, myList))
    print(myList)
    [2, 3, 5, 0]
    ```

2. `filter()` function with `lambda` function
    ```python
    myList = [2, 1, 3, 5, 1, 1, 1, 0]
    result = filter(lambda val: val != 1, myList) 
    print(list(result))
    [2, 3, 5, 0]
    ```

### 方法二(推荐)

**使用`列表推导式`从列表中删除元素的所有实例**

```python
myList = [1, 2, 3, 4, 2, 2, 3]
valueToBeRemoved = 2

myList = [value for value in myList if value != valueToBeRemoved]

print(myList)
[1, 3, 4, 3]
```

### 方法三

**使用`remove()`函数从Python列表中删除元素的所有实例**

```python
myList = [2, 1, 3, 5, 1, 1, 1, 0]
valueToBeRemoved = 1
 
try:
    while True:
        myList.remove(valueToBeRemoved)
except ValueError:
    pass
 
print(myList)

[2, 3, 5, 0]
```
