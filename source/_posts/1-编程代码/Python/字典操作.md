## 修改字典的键

{% note success%}
⚡️ 只需一行 `dic[“c”]=dic.pop（“a”` 即可直接将值弹出到新键
{% endnote %}

事实上，在Python中，字典的键**不能直接修改**，**因为键是经过哈希处理的**。

如果需要修改，可以使用一些技巧间接完成。

### 方法一(推荐)

弹出要修改的键的值并将其分配给新键
```python
dic = {'a':1, 'b':2}

dic['c'] = dic.pop('a')
```

### 方法二

与方法一类似，使用 `update` 方法
```python
dic = {'a':1, 'b':2}
 
dic.update({'c': dic.pop("a")})
```

### 方法三

首先，将旧键值分配给新键，然后删除旧键。
```python
dic = {'a':1, 'b':2}
 
dic['c'] = dic['a']
 
del dic['a']
```

## 根据值获取键

```python
s = {'a':100,'b':200,'c':300}
```
### 方法一

遍历提取键值对，判断值并提取键。
```python
keys = [k for k,v in s.items() if v==200]
```

### 方法二

将字典的值转换为列表，找到目标的索引，将键转换为列表，根据前面的索引获取值。
```python
keys = list(s.keys())[list(s.values()).index(200)]
```

### 方法三(不推荐)

新建一个字典，查询字典中的值，但有时值会重复，转成字典后只有一个
```python
new_dict={v:k for k,v in s.items()}
print(new_dict)
print(new_dict[200])
```

## 将两个列表转换为字典

{% note success%}
😎 直接用 `dict(zip(list_a, list_b))` 解决，其中 `list_a` 是**键**，`list_b` 是**值**。
{% endnote %}

### 方法一

使用 `zip()` 和 `dict()` 将两个列表转换为字典。
```python
key_list = ['name', 'age', 'address']
value_list = ['Johnny', '27', 'New York']

dict_from_list = dict(zip(key_list, value_list))
print(dict_from_list)

>>> {'name': 'Johnny', 'age': '27', 'address': 'New York'}
```

### 方法二

使用字典解析将列表转换为字典。
```python
key_list = ['name', 'age', 'address']
value_list = ['Johnny', '27', 'New York']

dict_from_list = {k: v for k, v in zip(key_list, value_list)}
print(dict_from_list)

>>> {'name': 'Johnny', 'age': '27', 'address': 'New York'}
```

如果两个列表的长度相同，则可以使用 `range()` 来遍历字典解析

```python
key_list = ['name', 'age', 'address']
value_list = ['Johnny', '27', 'New York']

dict_from_list = {key_list[i]: value_list[i] for i in range(len(key_list))}
print(dict_from_list)

>>> {'name': 'Johnny', 'age': '27', 'address': 'New York'}
```

### 方法三

使用 `for` 循环将列表转换为字典。

```python
key_list = ['name', 'age', 'address']
value_list = ['Johnny', '27', 'New York']

dict_from_list = {}
for key in key_list:
  for value in value_list:
    dict_from_list[key] = value
    value_list.remove(value)
    break

print(dict_from_list)

>>> {'name': 'Johnny', 'age': '27', 'address': 'New York'}
```

另一个使用循环的解决方案是使用 `range()` 迭代索引并检索键值对，就像字典理解的例子一样。

```python
key_list = ['name', 'age', 'address']
value_list = ['Johnny', '27', 'New York']

dict_from_list = {}
for i in range(len(key_list)):
  dict_from_list[key_list[i]] = value_list[i]

print(dict_from_list)

>>> {'name': 'Johnny', 'age': '27', 'address': 'New York'}
```
